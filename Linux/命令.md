# 命令

```markdown
1、查找文件 
	find / -name 文件名
2、当前路径
	pwd
3、切换路径
	cd 路径
4、删除文件
	rm 文件名
5、显示当前路径下所有文件
	ls
6、打开vim
	vim
7、用vim创建一个文件 —— 会保存在当前路径下
	vim 文件名
8、关于cat命令：
	8.1 显示整个文件
		cat 文件名
	8.2 创建一个文件
		cat >文件名	——注意：用这种方式创建的文件，以Ctrl + D结束
		cat <<EOF>> 文件名	——注意：用这种方式创建的文件，以某一行键入EOF结束
	8.3 连接多个文件内容到某个文件
		cat 文件名1 文件名2 >文件3	——注意：前面不止2个文件也可以，后面文件可以为新文件（即被cat命令创建了），也可以是已有的文件
```
